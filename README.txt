Запускать exe/dist/main/main.exe



При запуске игры появляется окно Меню, в котором: 
Управление стрелками вверх, вниз
Буква е(латиница) для выбора всего, кроме запуска лвла, для него нужно нажать f
 
Окно Меню:
Первое окно:
Start - Переключает на другое окно с выбором уровня
Quit - Выходит из игры
Второе окно:
Все кнопки кроме Back:
Открывают окно с уровнем
Back - Возвращает на первое окно

Окно игры:
Управление Стрелки(Вверх и вниз на лестницах), Пробел(Для прыжка), буква е(латиница) для взаимодействия с объектами
Дается три жизни
При утрате всех жизней либо при прохождении(дохода до флага):
Открывается окно конца игры

Окно конца игры:
Restart - Перезапускает Уровень
Back - Возвращение к окну Меню
Также пишется кол-во полученных монет при прохождении уровня, и после нажатия на одну из кнопок создается файл с название уровня и полученными монетами в последний раз прохождения
